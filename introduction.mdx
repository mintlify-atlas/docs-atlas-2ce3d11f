## What is create-nextjs-dapp?

create-nextjs-dapp is a command-line tool that helps you quickly scaffold production-ready Next.js decentralized applications (dApps) with your choice of wallet provider and blockchain. Inspired by [create-next-app](https://github.com/vercel/next.js/tree/canary/packages/create-next-app), it provides an opinionated, batteries-included setup that lets you start building immediately.

<Card title="Quick Start" icon="rocket" href="/quickstart">
  Get up and running in under 2 minutes
</Card>

## Why create-nextjs-dapp?

Setting up a new dApp from scratch requires configuring multiple dependencies, wallet providers, blockchain connections, and best practices. create-nextjs-dapp eliminates this complexity by:

- **Saving time**: Go from idea to running app in minutes, not hours
- **Following best practices**: Pre-configured with TypeScript, Next.js 14+ App Router, and Tailwind CSS
- **Staying flexible**: Choose from 8 wallet providers and 2 blockchain ecosystems
- **Keeping current**: Automatic update notifications ensure you're using the latest version

## Supported Blockchains

### EVM (Ethereum Virtual Machine)

Build dApps for Ethereum and EVM-compatible chains including:
- Ethereum (Mainnet & Testnets)
- Polygon
- Base
- Arbitrum
- Optimism
- And many more...

### Solana

Build high-performance dApps on the Solana blockchain with native wallet adapter support.

## Quick Example

Get started with a single command:

<CodeGroup>
```bash npm
npx create-nextjs-dapp
```

```bash yarn
yarn create nextjs-dapp
```

```bash pnpm
pnpm create nextjs-dapp
```

```bash bun
bunx create-nextjs-dapp
```
</CodeGroup>

Or skip the prompts with CLI options:

```bash
npx create-nextjs-dapp my-dapp --chain evm --wallet rainbowkit
```

## What You Get

Every generated project includes:

- **Next.js 14+** with App Router
- **TypeScript** with strict mode enabled
- **Tailwind CSS** for styling
- **Pre-configured wallet provider** of your choice
- **Example components** for wallet connection and contract interaction
- **Custom hooks** for blockchain interactions
- **Dark mode support** with theme toggle
- **Environment variable setup** with `.env.example`
- **ESLint & TypeScript** configuration
