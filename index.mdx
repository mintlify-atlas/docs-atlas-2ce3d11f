  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 text-center">
    Quick start
  </h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-12 text-center max-w-3xl mx-auto">
    Get your dApp running in under a minute
  </p>

  <Steps>
    <Step title="Run the CLI">
      Execute the create-nextjs-dapp command in your terminal:
      
      <CodeGroup>
      ```bash npm
      npx create-nextjs-dapp my-dapp
      ```
      ```bash yarn
      yarn create nextjs-dapp my-dapp
      ```
      ```bash pnpm
      pnpm create nextjs-dapp my-dapp
      ```
      ```bash bun
      bunx create-nextjs-dapp my-dapp
      ```
      </CodeGroup>

      The CLI will interactively prompt you to select your blockchain and wallet provider.
    </Step>

    <Step title="Configure your environment">
      Navigate to your project and set up environment variables:
      
      ```bash
      cd my-dapp
      cp .env.example .env.local
      ```

      Add your API keys (RPC URLs, wallet provider keys, etc.) to `.env.local` based on your chosen provider.
    </Step>

    <Step title="Install and run">
      Install dependencies and start the development server:
      
      ```bash
      npm install
      npm run dev
      ```

      Your dApp will be running at `http://localhost:3000` with wallet connection already configured.
    </Step>
  </Steps>
</div>
