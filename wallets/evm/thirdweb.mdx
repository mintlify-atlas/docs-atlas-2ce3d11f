## Overview

Thirdweb is a **complete web3 development platform** that goes beyond wallet connections. It provides tools for smart contract deployment, embedded wallets, NFT management, token creation, gasless transactions, and account abstraction - all in one unified SDK.

**Best for**: Teams building full-stack web3 applications that need comprehensive tooling beyond just wallet connections.

## Key Features

- **Complete Platform** - Wallet connections, contracts, NFTs, tokens, and more
- **Embedded Wallets** - Email and social login with auto-created wallets
- **Multi-Chain Support** - EVM and Solana chains
- **Smart Contract Tools** - Deploy and interact with contracts
- **Account Abstraction** - Gasless transactions and smart accounts
- **NFT & Token Tools** - Mint, manage, and trade digital assets
- **In-App Purchases** - Fiat on-ramps integrated
- **Self-Contained SDK** - No additional chain dependencies needed
- **TypeScript First** - Excellent type safety and DX

## Configuration

### Environment Variables

Create a `.env.local` file:

```bash .env.local
NEXT_PUBLIC_THIRDWEB_CLIENT_ID=your_client_id_here
NEXT_PUBLIC_CONTRACT_ADDRESS=0x...
```

<Steps>
  <Step title="Get a Thirdweb Client ID">
    1. Visit [Thirdweb Dashboard](https://thirdweb.com/dashboard)
    2. Sign up or log in
    3. Go to **Settings** > **API Keys**
    4. Create a new API key
    5. Copy the **Client ID** (not the Secret Key)
  </Step>
  
  <Step title="Configure Project Settings">
    In the Thirdweb Dashboard:
    - Add your domain to allowed domains
    - Configure authentication methods
    - Set up billing (free tier available)
  </Step>
  
  <Step title="Add to your .env.local">
    ```bash
    NEXT_PUBLIC_THIRDWEB_CLIENT_ID=abc123...
    ```
  </Step>
</Steps>

<Warning>
  The Thirdweb Client ID is **required** for production. Without it, your app will use rate-limited public endpoints.
</Warning>

### Client Setup

The CLI generates a `lib/client.ts` file:

```tsx lib/client.ts
import { createThirdwebClient } from "thirdweb";

const clientId = process.env.NEXT_PUBLIC_THIRDWEB_CLIENT_ID || "";

if (!clientId) {
  console.warn(
    "Missing NEXT_PUBLIC_THIRDWEB_CLIENT_ID. Get one at https://thirdweb.com/dashboard"
  );
}

export const client = createThirdwebClient({
  clientId,
});
```

### Provider Setup

The CLI generates a minimal `Providers.tsx` component:

```tsx components/Providers.tsx
"use client";

import { ReactNode } from "react";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { ThirdwebProvider } from "thirdweb/react";
import { ThemeProvider } from "next-themes";

// Client is created in lib/client.ts
import "../lib/client";

const queryClient = new QueryClient();

const Providers = ({ children }: { children: ReactNode }) => {
  return (
    <ThemeProvider attribute="class" defaultTheme="dark" enableSystem>
      <ThirdwebProvider>
        <QueryClientProvider client={queryClient}>
          {children}
        </QueryClientProvider>
      </ThirdwebProvider>
    </ThemeProvider>
  );
};

export { Providers };
```

### Thirdweb Hooks

Use Thirdweb's React hooks:

```tsx
import { useActiveAccount, useActiveWallet } from "thirdweb/react";

const MyComponent = () => {
  const account = useActiveAccount();
  const wallet = useActiveWallet();

  return (
    <div>
      {account ? (
        <div>
          <p>Connected: {account.address}</p>
          <p>Wallet: {wallet?.id}</p>
        </div>
      ) : (
        <p>Please connect your wallet</p>
      )}
    </div>
  );
};
```

### Reading Contracts

Read data from smart contracts:

```tsx
import { useReadContract } from "thirdweb/react";
import { getContract } from "thirdweb";
import { sepolia } from "thirdweb/chains";
import { client } from "../lib/client";

const contract = getContract({
  client,
  chain: sepolia,
  address: process.env.NEXT_PUBLIC_CONTRACT_ADDRESS!,
});

const MyComponent = () => {
  const { data: greeting, isLoading } = useReadContract({
    contract,
    method: "function greeting() view returns (string)",
  });

  if (isLoading) return <p>Loading...</p>;

  return <p>Greeting: {greeting}</p>;
};
```

### Writing to Contracts

Send transactions to smart contracts:

```tsx
import { useSendTransaction } from "thirdweb/react";
import { prepareContractCall } from "thirdweb";
import { getContract } from "thirdweb";
import { sepolia } from "thirdweb/chains";
import { client } from "../lib/client";

const contract = getContract({
  client,
  chain: sepolia,
  address: process.env.NEXT_PUBLIC_CONTRACT_ADDRESS!,
});

const MyComponent = () => {
  const { mutate: sendTransaction, isPending } = useSendTransaction();

  const handleUpdate = () => {
    const transaction = prepareContractCall({
      contract,
      method: "function setGreeting(string memory _greeting)",
      params: ["Hello from Thirdweb!"],
    });

    sendTransaction(transaction);
  };

  return (
    <button onClick={handleUpdate} disabled={isPending}>
      {isPending ? "Updating..." : "Update Greeting"}
    </button>
  );
};
```

### Account Information

Get detailed account information:

```tsx
import { useActiveAccount, useWalletBalance } from "thirdweb/react";
import { sepolia } from "thirdweb/chains";
import { client } from "../lib/client";

const AccountInfo = () => {
  const account = useActiveAccount();
  
  const { data: balance } = useWalletBalance({
    client,
    chain: sepolia,
    address: account?.address,
  });

  if (!account) return null;

  return (
    <div>
      <p>Address: {account.address}</p>
      <p>Balance: {balance?.displayValue} {balance?.symbol}</p>
    </div>
  );
};
```

## Advanced Features

### Deploy Contracts

Deploy contracts directly from your app:

```tsx
import { deployContract } from "thirdweb";
import { sepolia } from "thirdweb/chains";
import { client } from "../lib/client";

const deployMyContract = async () => {
  const contractAddress = await deployContract({
    client,
    chain: sepolia,
    abi: [...],  // Your contract ABI
    bytecode: "0x...",  // Your contract bytecode
    constructorParams: [],
  });
  
  console.log("Deployed to:", contractAddress);
};
```

### NFT Minting

Mint NFTs with Thirdweb:

```tsx
import { useNFT, useContract } from "thirdweb/react";

const NFTComponent = () => {
  const contract = useContract("0x...");
  const { data: nft } = useNFT(contract, "0");

  return (
    <div>
      <img src={nft?.metadata.image} alt={nft?.metadata.name} />
      <p>{nft?.metadata.name}</p>
    </div>
  );
};
```

### IPFS Storage

Upload files to IPFS:

```tsx
import { upload } from "thirdweb/storage";
import { client } from "../lib/client";

const uploadFile = async (file: File) => {
  const uri = await upload({
    client,
    files: [file],
  });
  
  console.log("Uploaded to:", uri);
  return uri;
};
```

## Troubleshooting

### Client ID Missing

**Issue**: Console warning about missing Client ID

**Solution**:
1. Get a Client ID from [Thirdweb Dashboard](https://thirdweb.com/dashboard)
2. Go to Settings > API Keys
3. Create a new API key
4. Copy the **Client ID** (not Secret Key)
5. Add to `.env.local`:
   ```bash
   NEXT_PUBLIC_THIRDWEB_CLIENT_ID=abc123...
   ```
6. Restart dev server

### Rate Limiting

**Issue**: "Rate limit exceeded" errors

**Solution**: 
- Set your Client ID in `.env.local`
- Without a Client ID, you're using public rate-limited endpoints
- Free tier includes generous limits

### Connect Button Not Working

**Issue**: Connect button doesn't open modal

**Solution**:
1. Verify `ThirdwebProvider` wraps your app
2. Ensure client is created correctly
3. Check that component is client-side (`"use client"`)
4. Verify Client ID is valid

### Contract Interaction Fails

**Issue**: Contract reads/writes fail

**Solution**:
1. Verify contract address is correct
2. Check that chain matches where contract is deployed
3. Ensure wallet is connected
4. Verify contract ABI is correct
5. Check that function signature matches:
   ```tsx
   method: "function greeting() view returns (string)"  // Exact signature
   ```

### SSR Errors

**Issue**: Server-side rendering errors

**Solution**:
1. Use `"use client"` directive
2. Wrap Thirdweb hooks in client components
3. Check for window/document usage
